# 🦷 Dental Healthcare Management System

A modern Full Stack Dental Clinic Management System developed using React.js, Node.js, Express.js and MySQL.

---

## ✨ Features

- 🔐 Secure Login & Authentication
- 👨‍⚕️ Doctor Management
- 👤 Patient Management
- 📅 Appointment Booking
- 💊 Medicine Management
- 🏥 Department Management
- 🏢 Branch Management
- 💬 Testimonials
- 📰 Blog Management
- ❓ FAQ Management
- 📈 Admin Dashboard
- 📱 Responsive Design

---

## 🛠 Tech Stack

### Frontend
- React.js
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MySQL

backend/

## 🚀 Installation

```bash
git clone https://github.com/230493131013mihir/dental-project.git
```

Frontend

```bash
cd frontend
npm install
npm run dev
```

Backend

```bash
cd backend
npm install
npm start
```
